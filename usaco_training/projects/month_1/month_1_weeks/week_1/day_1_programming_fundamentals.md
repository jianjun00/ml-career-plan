- **Focus**: Problem-solving approach, algorithmic thinking methodology
- **Key Concepts**: 
  - Problem decomposition (break complex problems into smaller parts)
  - Algorithm design principles (step-by-step problem solving)
  - Efficiency analysis basics (how to measure algorithm performance)
  - Big O notation introduction (O(1), O(n), O(log n), O(n²))
- **Learning Tasks**:
  1. Watch the full lecture (51 minutes)

  3. Write examples of O(1), O(n), O(n²) operations

## 💻 **Practice Problems (5 problems)**

### **USACO Bronze Problems**
1. **Mix Milk** - Basic arithmetic and sorting
   - **Problem URL**: https://usaco.guide/current/bronze/intro-comp/
   - **Skills**: Basic operations, sorting small arrays
   - **Focus**: Problem understanding and implementation
   - **Learning Steps**:
     1. Read problem statement carefully
     2. Identify input format (three integers)
     3. Determine output format (sorted values)
     4. Write solution using simple sorting
     5. Test with sample cases
   - **Expected Time**: 15 minutes
   - **Solution Approach**: Read three integers, sort them, output sorted

2. **Diamond Collector** - Greedy approach
   - **Problem URL**: https://usaco.guide/current/bronze/sort-search/
   - **Skills**: Array processing, greedy selection
   - **Focus**: Simple algorithm design
   - **Learning Steps**:
     1. Understand diamond sorting rules
     2. Read input array of diamonds
     3. Implement greedy selection algorithm
     4. Test with sample cases
   - **Expected Time**: 20 minutes
   - **Solution Approach**: Sort diamonds by size, apply greedy selection

### **Codeforces 800 Difficulty**
3. **1500A: Going Home** - Basic implementation
   - **Problem URL**: https://codeforces.com/problemset/problem/1500/A
   - **Skills**: String processing, basic logic
   - **Focus**: Input handling and output formatting
   - **Learning Steps**:
     1. Read string input
     2. Process according to problem requirements
     3. Format output correctly
   - **Expected Time**: 15 minutes
   - **Solution Approach**: Simple string processing with specific rules

4. **1501A: Alexey and Train** - Simple calculations
   - **Problem URL**: https://codeforces.com/problemset/problem/1501/A
   - **Skills**: Mathematical computations
   - **Focus**: Precision and accuracy
   - **Learning Steps**:
     1. Read input values
     2. Perform required calculations
     3. Output result with correct formatting
   - **Expected Time**: 15 minutes
   - **Solution Approach**: Basic arithmetic with input validation

### **AtCoder ABC Problems**
5. **ABC 285 A** - Basic arithmetic
   - **Problem URL**: https://atcoder.jp/contests/abc285/tasks/abc285_a
   - **Skills**: Basic operations, input handling
   - **Focus**: Problem interpretation
   - **Learning Steps**:
     1. Read two integers
     2. Perform required operation
     3. Output result
   - **Expected Time**: 10 minutes
   - **Solution Approach**: Simple arithmetic operation

### **Problem-Solving Strategy for Each Problem**
1. **Read Carefully**: Understand all requirements and constraints
2. **Plan Solution**: Think through approach before coding
3. **Implement**: Write clean, efficient code
4. **Test**: Verify with sample cases
5. **Submit**: Submit and check results
6. **Review**: Analyze official solution if available
